Help & User Guide

Getting Started

  • Navigate to the Scanner tab to upload and scan files, URLs, or IPs.
  • Choose the scan mode and output format (CSV, PDF, DOCX, or DB).
  • Click Run to start the scan.
  • Check the Reports section to view and download generated reports.
  • Use the Settings tab to adjust scan parameters like delay, headless mode, etc.

Troubleshooting Tips

  • If scans aren't working, make sure you're connected to the internet and VPN (if required).
  • Check your VirusTotal API key settings in the config file if using the API version.
  • Ensure the `input-files` and `generated-reports` folders exist and have proper permissions.
  • Refer to console logs for detailed error messages.
